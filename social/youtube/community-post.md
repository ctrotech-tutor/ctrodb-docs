Just shipped ctrodb v1.0 — a client-side database for TypeScript that runs in the browser and Node.js.

What it does:
- Schema validation at write time
- MongoDB-like queries with a fluent builder
- IndexedDB persistence (in-memory fallback for Node)
- React hooks for reactive UIs
- FTS, relations, and validation plugins built in

Zero runtime dependencies. 2,400 lines of TypeScript. MIT license.

Try it in the browser: https://ctrodb.vercel.app/playground
GitHub: https://github.com/ctrotech-tutor/ctrodb
