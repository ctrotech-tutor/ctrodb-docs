# Building with ctrodb — A 5-Part Series

A deep-dive series on building real-world applications with ctrodb, the zero-dependency client-side database for TypeScript.

## Articles

1. **Your First ctrodb App** — Set up, schema design, CRUD, and basic queries
2. **Reactive UIs with ctrodb + React** — useQuery, useMutation, DatabaseProvider
3. **Client-Side Full-Text Search** — Inverted indexes, FTS plugin, combining search with filters
4. **Relations and Schema Design** — has_many, belongs_to, has_one, eager loading, validation patterns
5. **Production Ready** — Transactions, error handling, plugins, performance, testing
