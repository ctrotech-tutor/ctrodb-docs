---
title: Installation
description: How to install ctrodb
---

## npm

```bash
npm install ctrodb
```

## pnpm

```bash
pnpm add ctrodb
```

## yarn

```bash
yarn add ctrodb
```

## bun

```bash
bun add ctrodb
```

## Requirements

- Node.js 18+ (for Node.js usage)
- The library itself works in any modern browser or Node 18+
- TypeScript types are included — no `@types/ctrodb` needed
- React bindings are in `ctrodb/react`, no extra package

## CDN

```html
<script src="https://unpkg.com/ctrodb@latest/dist/index.global.js"></script>
```

The global name is `CtroDB`.
